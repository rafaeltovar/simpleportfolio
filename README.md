# SimplePortfolio

"SimplePortfolio" is a PHP software for create a little and simple portfolio.

NO DATABASE NEEDED!!

More details [here](http://www.rafaeltovar.info/simpleportfolio)!

+ Little theme system
+ Portfolio style
+ Bootstrap 3!!
